⚡ Connect to Supabase

Set up your database once — all surveys, responses and team members will be shared across every device and browser automatically.

1

Create a free Supabase project

Go to supabase.com → New Project. Name it ncwsc-platform.

2

Run this SQL in Supabase → SQL Editor

CREATE TABLE surveys ( id text PRIMARY KEY, title text, description text, category text, status text, color text, created text, questions jsonb DEFAULT '[]' ); CREATE TABLE responses ( id text PRIMARY KEY, survey_id text REFERENCES surveys(id) ON DELETE CASCADE, ts text, data jsonb ); CREATE TABLE team_members ( id text PRIMARY KEY, name text, email text, role text, initials text, color text ); ALTER TABLE surveys ENABLE ROW LEVEL SECURITY; ALTER TABLE responses ENABLE ROW LEVEL SECURITY; ALTER TABLE team_members ENABLE ROW LEVEL SECURITY; CREATE POLICY "public_all" ON surveys FOR ALL USING (true) WITH CHECK (true); CREATE POLICY "public_all" ON responses FOR ALL USING (true) WITH CHECK (true); CREATE POLICY "public_all" ON team_members FOR ALL USING (true) WITH CHECK (true);
3

Get your credentials

Go to Settings → API in your Supabase project. Copy the Project URL and anon public key below.
